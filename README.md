# Student Performance Analysis

## Objective
This is my Python assignment project where I analyze student performance data. The goal is to explore a CSV dataset of student scores and find patterns using basic data visualization and simple machine learning.

## Tools Used
- Python
- Jupyter Notebook
- pandas (for reading and handling the CSV data)
- matplotlib & seaborn (for making graphs and plots)
- scikit-learn (for basic ML modeling)

## Setup Instructions

Here is how to get this project running on your own computer.

### 1. Create a virtual environment
Open your terminal inside the project folder and run:
```bash
python -m venv .venv
```

### 2. Activate the virtual environment
If you are using Git Bash on Windows:
```bash
source .venv/Scripts/activate
```

If you are using PowerShell on Windows:
```powershell
.\.venv\Scripts\activate.ps1
```

*(For Mac/Linux users, you would use `source .venv/bin/activate` instead).*

### 3. Install the required libraries
Once your virtual environment is active, install the packages from the requirements file:
```bash
pip install -r requirements.txt
```

### 4. Open the Jupyter Notebook
To see the code and run the analysis, start Jupyter by running:
```bash
jupyter notebook
```
This will open up a page in your web browser. Just click on the notebook file to view my code.

## Notes
- The dataset used for this project is a CSV file.
- Any plots or charts generated by the notebook will be saved directly as images in this main project folder.
